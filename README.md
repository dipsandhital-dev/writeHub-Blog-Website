# WriteHub - MERN Stack Blog Application

## Project Overview
WriteHub is a modern, responsive blog application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). The frontend uses React hooks and React Router for routing, and the UI leverages ShadCN UI components for a clean and modern design. Users can read, create, update, and delete blogs, as well as interact through comments.

---

## Features
- User Authentication: Signup, login, JWT-based session management
- Blog Management: Create, read, update, delete blogs
- Comments Section: Add comments to blogs
- Responsive UI: Fully responsive using ShadCN UI components
- Smooth routing: React Router v6
- Modern Theme: Clean light/dark design

---

## Tech Stack
**Frontend:** React.js (Hooks), React Router v6, ShadCN UI, Axios  
**Backend:** Node.js, Express.js, MongoDB (Mongoose), JWT, Bcrypt  
**Tools:** VSCode, Postman, Git/GitHub
